ALCOM

VRChat Creator Companion (VCC) unofficial alternative app.

For more information, please refer to the official website:
  https://vrc-get.anatawa12.com/alcom/

Included items
================

- README.en.txt this file
- README.ja.txt Japanese version
- website.url link to the official website
- LICENSE license file

How to install
================

Windows
----------------
1. Access https://vrc-get.anatawa12.com/alcom/ and download the Windows installer
2. Run the downloaded installer
3. Follow the installer instructions to complete the installation
4. Start ALCOM
5. Follow the instructions of ALCOM to perform the initial settings of ALCOM.
5. The Installation is complete.

If you are already using VCC, the VCC settings will be inherited.
Also, VCC and ALCOM can coexist, and you do not need to uninstall VCC.

macOS
----------------
1. Access https://vrc-get.anatawa12.com/alcom/ and download the macOS disk image
2. Open the disk image
3. Drag and drop ALCOM.app in the disk image to the Applications folder
4. Start ALCOM
5. Follow the instructions of ALCOM to perform the initial settings of ALCOM.
6. The Installation is complete.

If you are using VPM CLI or vrc-get, ALCOM will inherit the repository settings.

Linux
----------------
1. Access https://vrc-get.anatawa12.com/alcom/ and download the macOS AppImage
2. Run the AppImage and start ALCOM
3. Follow the instructions of ALCOM to perform the initial settings of ALCOM.
4. The Installation is complete.

If you are using VPM CLI or vrc-get, ALCOM will inherit the repository settings.

The Linux version is built in the Ubuntu 22.04 environment.
