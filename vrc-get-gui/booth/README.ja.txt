ALCOM

VRChat Creator Companion (VCC) の非公式代替アプリです。

詳しい情報は公式サイトを参照してください:
  https://vrc-get.anatawa12.com/alcom/

インストール方法
================

Windows
----------------
1. https://vrc-get.anatawa12.com/alcom/ にアクセスし、 Windows インストーラーをダウンロードします
2. ダウンロードしたインストーラーを実行します
3. インストーラーの指示に従ってインストールを完了します
4. ALCOM を起動します
5. ALCOM の指示に従い、 ALCOM の初期設定を行います。
6. インストールは完了です。

VCCをすでに使用している場合には、 VCC の設定が引き継がれます。
また、 VCC と ALCOM は共存でき、 VCC をアンインストールする必要はありません。

macOS
----------------
1. https://vrc-get.anatawa12.com/alcom/ にアクセスし macOS ディスクイメージをダウンロードします
2. ディスクイメージを開きます
3. ディスクイメージ内の ALCOM.app をアプリケーションフォルダにドラッグアンドドロップします
4. ALCOM を起動します
5. ALCOM の指示に従い、 ALCOM の初期設定を行います。
6. インストールは完了です。

VPM CLI や vrc-get を使用してる場合には、 ALCOM はそれらのレポジトリ設定を引き継ぎます。

Linux
----------------
1. https://vrc-get.anatawa12.com/alcom/ にアクセスし macOS AppImage をダウンロードします
2. AppImage を実行し、 ALCOM を起動します
3. ALCOM の指示に従い、 ALCOM の初期設定を行います。
4. インストールは完了です。

VPM CLI や vrc-get を使用してる場合には、 ALCOM はそれらのレポジトリ設定を引き継ぎます。

なお、 Linux 版は Ubuntu 22.04 環境でビルドされています。
